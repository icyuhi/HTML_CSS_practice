未経験から始めるプログラミング

プログラミングに触れたことのなかった自分が
HTMLとCSSから学んで行く過程をブログにまとめました。
このページは、HTMLとCSSのみで製作しています。

はてなブログで学習の進捗を読む

独学で進めているため更新頻度はかなり遅いです。
頑張って早くJSコーディングできるようにしたいです。　

独学する予定の言語一覧

    HTML

    マークアップ言語

    HTMLとは、
    Hyper Text Markup Languageの略で
    Head要素、Body要素、HTML要素の
    ３要素で出来ています。

    CSS

    マークアップ言語

    CSSとは、
    Cascading Style Sheetsの略で
    HTMLのタグで囲んだ情報に
    装飾を施しWebページのスタイルを
    指定するための言語です。

    Java Script

    プログラミング言語

    (フロントエンド)

    JavaScriptとは、
    Webサイト、Webアプリ、バックエンド、
    デスクトップアプリ、モバイルアプリなど、
    ブラウザからサーバ、デスクトップから
    スマートフォンまで広く使用される
    オブジェクト指向スクリプト言語です。
    Javaとは、異なるプログラミング言語。

    PHP

    プログラミング言語

    （サーバーサイド）

    PHPとは、
    Hypertext Preprocessorの略で
    HTMLに埋め込むことができる
    Web開発に適した汎用スクリプト言語です。
    実行するためには
    PHPがインストールされたWebサーバーが必要。
